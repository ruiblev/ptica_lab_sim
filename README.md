# Simulador de Ótica - Física e Química A (11.º Ano)

Este projeto consiste num simulador interativo em Python utilizando o framework **Streamlit**, desenvolvido com base no protocolo da **Atividade Laboratorial (AL)** para as turmas de Física e Química A do 11.º ano letivo.

## Funcionalidades
A aplicação está dividida em duas secções principais (abas):
1. **Reflexão, Refração e Reflexão Total**: Estuda a Lei da Reflexão e a Lei de Snell-Descartes para a refração, bem como a determinação do ângulo crítico para a reflexão total.
2. **Difração da Luz**: Estuda os máximos de interferência para feixes laser que passam por uma fenda simples ou rede de difração, relacionando o comprimento de onda do laser com a distância ao alvo e as propriedades da rede.

## Requisitos e Instalação

As dependências estão listadas no ficheiro `requirements.txt`:
```bash
pip install -r requirements.txt
```

## Executar o Simulador

Para correr a aplicação, execute o seguinte comando no terminal dentro da pasta do projeto:
```bash
streamlit run app.py
```

## Enquadramento Curricular (Portugal)
O simulador respeita as orientações da DGE e as Aprendizagens Essenciais para:
* Investigar, experimentalmente, os fenómenos de reflexão, refração, reflexão total e difração da luz.
* Determinar o índice de refração de um meio.
* Determinar o comprimento de onda da luz num laser usando redes de difração.
